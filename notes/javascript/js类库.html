# @antv/g6open in new window

G6 是一个图可视化引擎。它提供了图的绘制、布局、分析、交互、动画等基础的图可视化能力。旨在让关系变得透明，简单。让用户获得关系数据的 Insight。

$\color{red}{g6}$中，Graph对象是图的载体，包含了图上的所有元素，$\color{red}{即是说使用G6是在Graph对象之上进行扩展的}$

$\color{red}{G6的生命周期}$:初始化 —> 加载数据 —> 渲染 —> 更新 —> 销毁

<script>
    import G6 import '@antv/g6'
    export default{
        name: 'g6x',
        data(){
            return{
                initData: {
        		// 点集
        		nodes: [
          			{
            			id: 'node1', // 节点的唯一标识
            			x: 100, // 节点横坐标
            			y: 200, // 节点纵坐标
            			label: '起始点', // 节点文本
          			},
          			{
            			id: 'node2',
            			x: 300,
            			y: 200,
            			label: '目标点',
          			},
        		],
        			// 边集
        			edges: [
          			// 表示一条从 node1 节点连接到 node2 节点的边
          			{
           			source: 'node1', // 起始点 id
            			target: 'node2', // 目标点 id
            			label: '我是连线', // 边的文本
          			},
        		],
      		},
            }
        },
        mounted() {
    		this.initG6()
  	},
  	methods: {
    		initG6() {
      			const graph = new G6.Graph({
        		container: 'g6', // 指定挂载容器
        		width: 800, // 图的宽度
        		height: 500, // 图的高度
      			})
      			graph.data(this.initData) // 加载数据
      			graph.render() // 渲染
      		},
      	},
    }
</script>

# @antv/x6open in new window

X6 是 AntV 旗下的图编辑引擎，提供了一系列开箱即用的交互组件和简单易用的节点定制能力，方便我们快速搭建流程图、DAG 图、ER 图等图应用。
