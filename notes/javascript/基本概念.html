# 回调函数

把一个函数作为参数传递给另外一个函数，就称这个函数为回调函数。

function one(name){
	return name
}

function two(func){
	console.log("hi",func)
}
// 函数one()是two()的回调函数
two(one('charm'))

# 高阶函数（Higher-Order Function）

一个返回另外一个函数的函数被称为高阶函数

function one(name){
	return name
}

function two(func){
	console.log("hi",func)
    return function(){
        console.log('is so bad!')
    }
}
// two是一个高阶函数，bad()、two(one("charm"))调用了two的返回值
const bad = two(one("charm"))
bad()
two(one("charm"))()

# 原型编程

是一种 面向对象编程 的风格。在这种风格中，我们不会显式地定义类 ，而会通过向其它类的实例（对象）中添加属性和方法来创建类，甚至偶尔使用空对象创建类。 简单来说，这种风格是在不定义$\color{red}{class}$的情况下创建一个 $\color{blue}{object}$。
