  • Git

    • git init [warehouse name] 初始化仓库
    • git add [file name] 将文件添加到仓库
    • git commit -m "说明" 把文件提交到仓库并提供解释
    • git status 查看仓库当前状态
    • git diff 查看上次修改的内容
    • git log [--pretty=oneline] 查看提交日志
    • git reset --hard 版本号 指定某一版本(版本号)
    • git reflog 显示命令历史
    • git checkout -- [file name] 丢弃工作区的修改
    • git rm [file name] 删除一个文件
    • 关联本地仓库与远程仓库
    • git remote add [分支名] url 添加关联
    • git push -u [分支名] master 将本地库的所有内容推送到远程库，之后可以使用git push简化命令
    • git config --global credential.helper store免密push
    • git config --global core.quotepath false 中文乱码
